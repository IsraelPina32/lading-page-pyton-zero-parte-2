# Python + IA Landing Page - Asimov Academy

### 1. Ferramentas de IA Utilizadas
Utilizei o modelo **Gemini 3 Flash e Claude** como co-piloto de engenharia e o motor de estilização do **Tailwind CSS via CDN** para prototipação rápida.

### 2. Etapas de Maior Contribuição
A IA foi fundamental na **composição do layout (Bento Grid)** e na escrita da **cópia persuasiva**, acelerando o processo de design para desenvolvimento.

### 3. Ajustes Manuais e Motivação
Refinei manualmente a **paleta de cores** e a **interatividade dos CTAs** para alinhar o projeto às referências dos sites (Linear/Antimetal), garantindo uma experiência de usuário mais fluida e profissional,interativa e usuabilidade.
